ClearConnect

an open-source micro services platform written in Java.

Applications built with ClearConnect instantly get these out-of-the-box features:

  • real-time data transfer
  • high-availability
  • real-time monitoring and data inspection

ClearConnect makes developing distributed applications quick and easy by providing a simple API and tools.
